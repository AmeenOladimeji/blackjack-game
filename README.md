# blackjack-game
Python Object-Oriented Blackjack

A terminal-based implementation of the classic casino game Blackjack built from scratch using clean Object-Oriented Programming principles in Python.

Features include a modular Card and BlackjackGame class design managing game state seamlessly. The program supports standard actions including Hit, Stand, and Double Down. It features a smart dealer AI that automatically manages dealer hits and holds strictly on rules by stopping at 17. The program also provides dynamic visuals with a custom ASCII layout that renders the playing cards beautifully directly in the terminal console.

To run this project ensure you have Python 3 installed then run the script via your terminal.
